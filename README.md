<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Having a C compiler (gcc) and the capacity to use make from a Makefile

### Installation

1. Clone the repo
```sh
git clone https://github.com/nicklamyeeman/CPool_Tree_2017.git
```
2. Compile it
```sh
make
```
